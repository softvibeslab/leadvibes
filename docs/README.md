## What is LeadVibes?

LeadVibes is a multi-tenant CRM platform that combines:

- **Lead Management**: Track leads through the sales pipeline
- **Gamification**: Points system and leaderboards to motivate brokers
- **AI Assistant**: Chat interface for sales coaching
- **Activity Tracking**: Log all broker interactions
- **Dashboard**: Real-time statistics and performance metrics

## Key Features

- 🏢 **Multi-tenant Architecture**: Isolated data per agency
- 🎯 **Lead Pipeline**: Track from new lead to closed sale
- 🏆 **Gamification**: Points, badges, and leaderboards
- 🤖 **AI Coaching**: AI assistant for sales guidance
- 📊 **Analytics**: Real-time dashboard and reporting
- 📱 **Responsive**: Works on desktop and mobile

## Technology Stack

- **Backend**: Python 3.13+, FastAPI, MongoDB 8
- **Frontend**: React 19, TailwindCSS, Radix UI
- **AI/LLM**: LiteLLM with OpenAI/Claude/Google support
- **DevOps**: Docker, Docker Compose
