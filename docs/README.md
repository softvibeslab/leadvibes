## Metodología

Esta estrategia sigue el método de:

1. **Pilares**: 4 grandes áreas temáticas
2. **Tópicos**: 4 sub-áreas por pilar
3. **Secciones**: Agrupaciones de temas periódicos
4. **Formatos**: Carrusel, Reel, Imagen Simple
5. **Prueba**: 3 posts por sección para validar
6. **Seguimiento**: Documentación de resultados
